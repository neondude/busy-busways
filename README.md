## Getting Started
- Install Node v14
- run ```npm install``` in the project directory
- add google maps api key to the ```VITE_APP_GOOGLE_MAP_KEY``` property in either ```.env``` file or create a new file ```.env.local``` and add it there
- ```npm run dev```
